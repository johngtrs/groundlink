# GroundLink

GroundLink is a platform connecting emerging metal bands and underground venues.  
Built with Laravel (Backend) + React (Frontend) + Docker (Deployment).

## Tech Stack
- PHP 8.3 (Laravel 11)
- React 19 (Vite + MUI)
- Node 20
- MySQL 8
- Docker Compose

## Installation

```bash
docker-compose up --build
